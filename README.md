# Hello DevOps with Java :coffee:

This is a simple **"Hello DevOps"** done with **Java** programming language.

## Source code

This is the source code of the program:

```java
public class HelloDevOps {

    public static void main(String[] args) {

        System.out.println("===================================");
        System.out.println(" Welcome to DevOps Crash Course ");
        System.out.println(" Hello DevOps!");
        System.out.println("===================================");

    }
}
```

Notice that `System.out.println("Hello DevOps!");` shows the string `"Hello DevOps!"` on the screen.

## Compile program

To compile the "Hello World" program, type the following:

```console
javac HelloDevOps.java
```

## Excute the program

To execute the program, type this:

```console
java HelloDevOps
```
